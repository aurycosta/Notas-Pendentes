Como usar
1) Extraia o ZIP
2) Abra a pasta no VSCode e use o Live Server (ou qualquer servidor local)
3) Clique em "Importar XLSX" e selecione sua planilha

Observações
- Esse XLSX NÃO possui linha de cabeçalho (o arquivo já começa com dados). O painel lê por posição das colunas.
- O projeto usa bibliotecas via CDN: XLSX, Chart.js, html2canvas e jsPDF.
  Se você quiser versão 100% offline (sem internet), me peça que eu te gere um ZIP com as libs embutidas.
